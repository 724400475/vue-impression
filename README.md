# vue-impression
A Vue.js 2.0 UI elements for mobile.

## Installation
```shell
    npm i vue-impression -save
```

## Usage
components:
```javascript
    import Vue from 'vue'
    import Impression from 'vue-impression';

    Vue.use(Impression);
```

scss:
```scss
    @import '../../node_modules/vue-impression/dist/styles/index';
    @import 'myProjectCss';
```

## Example
```html
    <btn theme="primary" size="sm" @click="doSomething">按钮</btn>
```
